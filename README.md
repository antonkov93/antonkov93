## 🤵 Обо мне

Я начинающий QA-инженер, для которого важно, чтобы продукт работал безупречно. Не могу пройти мимо несоответствия, бага или неточности - эта особенность характера идеально подходит для тестирования, где важна дотошность и стремление к идеальному результату. Максимально мотивирован к профессиональному росту: каждый день изучаю что-то новое, слежу за актуальными практиками в сфере тестирования, открыт к любой обратной связи.

### 1. 🍕 Автоматизированное тестирование веб-приложения пиццерии [(ССЫЛКА)](https://pizzeria.skillbox.cc/)

**Выполненные задачи:**
- 📝 Написаны [тестовые сценарии](https://docs.google.com/document/d/1-_aV9O_LzD2yqtlq3fRJMLHrGe12YJ5cPPOMv59_gy0/edit?tab=t.0#heading=h.23nejaknpbek) для основного пользовательского флоу

- 📊 Настроено логирование действий с выводом в консоль и файл
- 📈 Настроены Allure-отчёты для визуализации результатов тестирования

**Технологии:** Python, Pytest, Selenium, Page Object Model, Allure Report, Git, flake8, Chrome DevTools
